# Github 

> * Iniiciar um projeto quase nunca repetir
```
git init
```

> * Adicionar os elementos na area de stage
```
git add .
```

> * Se usar o git init criar um repo
```
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/gsnegocios/first-projete.git
git push -u origin main
```

> * Clonar um projeto
```
git clone 
```
